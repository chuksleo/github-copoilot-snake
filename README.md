# Snake Game

A classic Snake game built with HTML, CSS, and JavaScript. Control the snake, eat food to grow, and avoid obstacles. The game ends if the snake collides with itself or an obstacle.

## Features
- **Classic Gameplay**: Control the snake using arrow keys.
- **Dynamic Scoring**: Earn 50 points for each food eaten.
- **Obstacles**: Avoid obstacles to keep the game going.
- **Game Over Screen**: Displays your score when the game ends.
- **Custom Design**: Includes a gradient background, glowing snake, and styled food.

## How to Play
1. Use the arrow keys (`↑`, `↓`, `←`, `→`) to control the snake.
2. Eat the food to grow and increase your score.
3. Avoid colliding with the walls, obstacles, or yourself.
4. The game ends when a collision occurs.

## How to Run
Follow these steps to run the game locally:

### Prerequisites
- A modern web browser (e.g., Chrome, Firefox, Edge).
- A code editor (optional, for modifications).

### Steps
1. **Download the Files**:
   - Ensure you have the following files in the same directory:
     - `index.html`
     - `snake.js`
     - Any sound files (e.g., `food.mp3`, `gameover.mp3`) if added.

2. **Open the Game**:
   - Double-click the `index.html` file to open it in your browser.

3. **Start Playing**:
   - The game will load, and you can start playing immediately.
